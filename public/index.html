🚀 Job Claimer API

MongoDB-Powered Job Management API

🟢 API Online

📋 About

A modern, scalable REST API for job claiming and management. Built with Express.js and MongoDB, featuring JWT authentication, role-based access control, and file upload capabilities.

✨ Features

🔐 Secure Authentication

JWT-based authentication with bcrypt password hashing

👥 User Management

Admin and member roles with granular permissions

📁 File Uploads

PDF submission support with validation

⚡ Fast & Scalable

MongoDB-optimized for performance

🔌 API Endpoints

Authentication

POST /api/users/register

Register a new user account

POST /api/users/login

Login and receive JWT token

GET /api/users/me

Get current authenticated user (requires token)

GET /api/users

Get all users (admin only)

PUT /api/users/:id

Update user details (admin only)

DELETE /api/users/:id

Delete a user (admin only)

Jobs Management

GET /api/jobs

Get all jobs with optional filtering (status, category, priority, search)

GET /api/jobs/:id

Get a specific job by ID

POST /api/jobs

Create a new job (admin only)

PUT /api/jobs/:id

Update job details (admin only)

POST /api/jobs/:id/claim

Claim a job for yourself

POST /api/jobs/:id/unclaim

Unclaim a job

POST /api/jobs/:id/submit

Submit work for a claimed job (includes PDF file upload)

DELETE /api/jobs/:id

Delete a job (admin only)

Health & Status

GET /

API health check and status

GET /health

Detailed health check with database status

🛠️ Getting Started

To integrate with this API:

  1. Register a user account via /api/users/register
  2. Login to receive your JWT token via /api/users/login
  3. Include the token in the Authorization header: Bearer YOUR_TOKEN
  4. Make authenticated requests to the API endpoints
